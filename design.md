# Suture Education Webapp - Design Style Guide

## Design Philosophy

### Visual Language
- **Medical Precision**: Clean, clinical aesthetic reflecting surgical environment
- **Educational Clarity**: Information hierarchy that guides learning progression
- **Professional Authority**: Trustworthy design conveying medical expertise
- **Interactive Engagement**: Modern, intuitive interface encouraging exploration

### Color Palette
- **Primary**: Deep Teal (#1B4B5A) - Medical professionalism, trust
- **Secondary**: Warm Coral (#E07A5F) - Accent for interactive elements
- **Tertiary**: Sage Green (#81B29A) - Natural healing, tissue colors
- **Neutral Base**: Off-white (#FAFAFA) - Clean medical background
- **Text**: Charcoal (#2C3E50) - High contrast readability

### Typography
- **Display Font**: "Tiempos Headline" - Bold, authoritative headings
- **Body Font**: "Suisse Int'l" - Clean, medical-grade readability
- **Monospace**: "JetBrains Mono" - Technical data and measurements

## Visual Effects & Styling

### Core Libraries Integration
1. **Anime.js**: Smooth micro-interactions and state transitions
2. **ECharts.js**: Medical data visualization and comparison charts
3. **p5.js**: Interactive anatomical diagrams and needle animations
4. **Splide.js**: Suture material carousels and image galleries
5. **Matter.js**: Physics-based suture tension demonstrations
6. **Shader-park**: Subtle background effects mimicking tissue textures
7. **PIXI.js**: High-performance visual effects for complex interactions

### Animation Strategy
- **Entrance Animations**: Subtle fade-ins with slight vertical translation
- **Hover States**: Gentle scale transforms and color transitions
- **Loading States**: Medical-inspired pulse animations
- **Data Transitions**: Smooth chart updates and filter animations
- **Scroll Triggers**: Progressive content revelation

### Header Effects
- **Background**: Subtle shader-generated organic patterns
- **Typography**: Large, bold headings with gradient text effects
- **Interactive Elements**: Floating suture icons with physics simulation

### Layout System
- **Grid**: 12-column responsive grid with medical proportions
- **Spacing**: Consistent 8px base unit system
- **Containers**: Maximum width constraints for optimal readability
- **Breakpoints**: Mobile-first responsive design

### Component Styling
- **Cards**: Clean white backgrounds with subtle shadows
- **Buttons**: Rounded corners with medical-inspired colors
- **Forms**: Clinical precision with clear validation states
- **Tables**: Alternating row colors for data clarity
- **Charts**: Medical color palette with clear data visualization

### Interactive Elements
- **Suture Selection Tool**: Multi-step wizard with progress indicators
- **Comparison Matrix**: Sortable tables with visual property indicators
- **Needle Explorer**: 3D-style interactive needle representations
- **Filter Controls**: Toggle switches and dropdown menus

### Background Design
- **Primary**: Consistent off-white base throughout
- **Texture**: Subtle organic patterns using shader effects
- **Depth**: Layered visual elements creating spatial hierarchy
- **Focus**: Clean areas for content concentration

### Medical Imagery Style
- **Photography**: High-resolution clinical images
- **Illustrations**: Clean, technical medical drawings
- **Icons**: Minimalist medical symbols and indicators
- **Charts**: Data-driven visualizations with medical context

### Accessibility Features
- **Contrast**: Minimum 4.5:1 ratio for all text
- **Focus States**: Clear keyboard navigation indicators
- **Screen Readers**: Proper ARIA labels and descriptions
- **Motion**: Respects user motion preferences

### Mobile Optimization
- **Touch Targets**: Minimum 44px for all interactive elements
- **Typography**: Responsive scaling for optimal readability
- **Navigation**: Collapsible menu system
- **Content**: Prioritized information hierarchy

## Brand Integration
- **Logo**: Clean, medical-inspired typography
- **Icons**: Consistent medical symbol library
- **Colors**: Professional medical palette throughout
- **Voice**: Authoritative yet approachable educational tone
